Compass是什么？

Compass是Sass的工具库

安装

gem install compass

使用

compass create myproject

编译

compass compile

--output-style参数

compass compile --output-style compressed

--force参数

compass compile --force

compass watch命令

compass watch

Compass的模块

  • reset
  • css3
  • layout
  • typography
  • utilities

reset模块

@import "compass/reset";

End
